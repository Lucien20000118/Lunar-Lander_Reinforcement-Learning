# 強化式學習專案

## 環境
- 基於 OpenAI Gym 開發的 Lunar Lander Environment
- 新增 Action (增加了頭頂噴射)
- 增加障礙物
- 更改指定地點
- 更改目標

## 技術
使用 PPO (Proximal Policy Optimization)

## 目標
讓 Lunar Lander 在不撞上障礙物的情況下，安全地降落在指定平台中，並穩定。
